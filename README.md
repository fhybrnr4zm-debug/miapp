# miapp - NBA Explorer

Aplicación web sencilla para explorar los 30 equipos de la NBA.

## Funcionalidades

- Buscar equipos por nombre o ciudad.
- Filtrar por conferencia (Este / Oeste).
- Ver detalles de cada equipo: conferencia, división, arena, año de fundación y campeonatos.

## Archivos

- `index.html`: estructura de la página.
- `style.css`: estilos visuales.
- `script.js`: datos de los equipos y lógica de la aplicación.

## Cómo usarla

Abre `index.html` en tu navegador, o publica el repositorio con GitHub Pages.
